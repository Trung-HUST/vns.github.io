# Các tùy chọn cấu hình

Phần này liệt kê các tùy chọn cấu hình giúp kiểm soát cách hệ thống, ứng dụng hoặc thiết bị hoạt động theo nhu cầu của người dùng.

**1)** Để thêm các tùy chọn cấu hình, hãy đi tới bảng quản trị và chọn **Cấu hình >> Nội dung**. Tiếp theo, bạn sẽ thấy các tùy chọn bên dưới:

**Tùy chọn so sánh** - Cho phép bạn so sánh sản phẩm dễ dàng.

**Tùy chọn danh sách yêu thích** - Thêm tính năng danh sách yêu thích.

**Tùy chọn tìm kiếm bằng hình ảnh** - Giúp tìm sản phẩm từ hình ảnh.

<ImagePopup src="/images/configure/options.png" alt="Các tùy chọn cấu hình" />

Sau khi chọn các tùy chọn mong muốn, hãy nhấp vào nút **Lưu cấu hình**.
